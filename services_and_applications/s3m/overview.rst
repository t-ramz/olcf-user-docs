What is S3M (OLCF API)?
=======================

S3M is an OLCF facility-managed gateway which aims to provide access to a wide range of OLCF API services.
This is done via OLCF-Project level access tokens which can then be used with an OLCF-hosted API.

.. note::
    S3M Currently only supports OLCF Open Projects

S3M Use Cases
-------------
* Submit Slurm jobs programmatically
* Set up data streaming services for OLCF compute resources
* View OLCF center/resource statuses programmatically
